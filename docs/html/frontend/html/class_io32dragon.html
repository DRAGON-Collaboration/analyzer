uint32_t Io32dragon::get_scaler_route ( HNDLE  odbHandle,
const char *  scalerpath,
INT *  status 
)

Read scaler routing from the ODB.

Scaler routing is set in four groups of four input channels each. Each group is defined by a series of 4 bits of register 4*17 starting with 0, i.e. [3..0], [7..4], [8..11], and [15..12]. The 4-bit number assigned to each group sets a block of four input channels. Input channel assignments are done in the following series, counting in blocks of 4:

  • z[15..0],FP_NIMOUT[15..0],LVDS_IN[15..0],NIM_IN[15..0] [n.b.: last <<– first ] For example to set ECL inputs [7..4] as the 2nd block, you would write the number 5 to bits [7..4] of the register.

Here we read the block numbers from ODB parameters and then set the appropriate bits and return the result.

int Io32dragon::init ( HNDLE  odbHandle)

Initialize the IO32.

Set up the io32 FPGA. Some settings are different depending on head or tail. Handled by conditionals on the global FRONTEND_TYPE variable.

Set up scaler routing, see io32_get_scaler_route()

Only enable 1st 16 scaler chanels (plus clock channel - 31) as the rest aren't available in the firmware. Register 62 handles this by writing a 0 to every channel to be enabled.

Note
static global 'fNumScalers' set here

◆ read_scaler()

INT Io32dragon::read_scaler ( char *  pevent)

Read scalers from the IO32.

Scalers are read out in three banks:

  • Head Bank Name (Tail Bank Name): Description
  1. "SCHD" ("SCTD"): Total number of counts in the current read [TID_WORD].
  2. "SCHR" ("SCTR"): Calculated scaler rate in the current read [TID_DOUBLE].
  3. "SCHS" ("SCTS"): Calculated scaler sum in the current run [TID_DWORD].

void Io32dragon::read_tsc4 ( char *  pevent)

Read TSC data.

Entries in the TSC FIFO are 32 bits. The leading two bits tell which IO32 input channel is being read, and the trailing 30 give the timestamp value.

Note
See init_vme_modules() to see where the signal routing is set for each IO32.
