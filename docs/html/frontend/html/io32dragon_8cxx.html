[in]in_clockInput channel for external clock
[in]clock_optionClock selector: 0 = internal 20MHz, 1 = reserved, 2 = external clock from selected input, 3 = external clock from selected input divided by 2 to form 50% duty cycle (for use with the 40 MHz V1190 clock).

Channel inputs: 5 bits starting w/ bit n*5 for channel n

External clock input: bits 24..20

Clock selector: bits 31..30
