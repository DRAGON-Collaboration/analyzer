void rbdragon::SetAutoZero ( Int_t  level = 1)

Turn on/off histogram auto-zeroing at run start.

Parameters
levelInteger specifying auto zero level:
  • 0: No auto zeroing
  • 1: Auto zero scalers only
  • 2: Auto zero all histograms
Note
The auto zero level can be controlled for online analysis by modifying the "/dragon/rootbeer/AutoZero" ODB variable. Set this variable to the desired auto zeroing level as explained for the level parameter.
