template<class T>
struct dragon::utils::less_and_valid< T >

Functor for sorting in ascending order, taking into account validity.

Overloads operator() such that invalid values are always considered "smaller" than any others.

Example:

double data[5] = { -1., 12., 11., -13., -1. }
std::sort(data, data+5, utils::less_and_valid<double>());
// data = { -1., -1., -13., 11., 12. }
