void Sonik::calculate ( )

Performs energy and time calibrations.

Does a linear transformation on each element in this->ecal[] using the slopes and offsets from variables.adc_slope and variables.adc_offset, respectively. Also calibrates the TDC signal; calculates ehit, hit.
