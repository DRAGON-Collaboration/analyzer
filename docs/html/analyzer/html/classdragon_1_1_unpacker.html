Switch over to coincidence mode

If in singles mode when called, it switches over to coincidence mode with default coincidence window and queue time. To change these values, use SetCoincWindow() and SetQueueTime().

If we are already in coincidence mode, this has no effect.

void dragon::Unpacker::SetSinglesMode ( int  qFlush = -1)
inline

Put the unpacker in singles mode

Any incoming events after this call are processed as singles only. If there are events in the queue, they are first flushed.

Parameters
qFlushMaximum flush time for events in the queue. Specifying a 0 skips flushing entirely and throws away any events in the queue. Negative numbers cause the entire queue to be flushed.
