UDouble_t dragon::MeasurementWeightedAverage ( InputIterator  begin,
InputIterator  end 
)

Calculate weighted average of measurements.

Procedure:

  1. Calculate num = \Sum_i { x_i * (1./sigma_i^2) }
  2. Calculateden= { 1./sigma_i^2 }
  3. Final result: nominal value = num/den, error = sqrt(den)
Attention
Assumes symmetric errors.
