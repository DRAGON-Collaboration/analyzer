Detailed Description

Helper class for lab -> CM conversions (fully relativistic)

Calulations are fully relativistic: For a moving beam (1) and stationary target (2): Ecm^2 = m1^2 + m2^2 + 2*m2*E1 or in terms of kinetic energy: (Tcm + m1 + m2)^2 = m1^2 + m2^2 + 2*m2*(m1 + T1)
