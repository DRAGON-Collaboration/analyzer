Detailed Description

Provides functions for setting and checking if data fields are invalid.

Author
G. Christian

Since ROOT trees require a valid entry for every parameter in every event, we use a "throw away" value to signify when a particular piece of data wasn't present in an event. In this file, the throw away value is set, and some functions are provided to check if data fields are equal to the value, and to set fields equal to the value.

Note
Most of the template functions were meta-programmed using utils/gen.py
