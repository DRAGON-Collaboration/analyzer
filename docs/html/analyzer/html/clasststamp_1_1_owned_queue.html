Detailed Description

template<class T>
class tstamp::OwnedQueue< T >

Queue that is a member of another class which handles popped events.

The intended use of this class is for when the queue exists as a data member of some other class (the "owner"). The owning class will perform the work of handling singles or coincidence events popped from the queue (see below).

Template Parameters
TThe type of class owning the queue; must have the following member functions:
  • void Process(const midas::Event&);
  • void Process(const midas::Event&, cont midas::Event&);
  • void Process(tstamp::Diagnostics*);

to handle singles and coincidence events, respectively.
