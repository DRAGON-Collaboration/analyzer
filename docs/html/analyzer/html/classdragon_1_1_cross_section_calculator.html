Calculate cross sections for runs in fBeamNorm.

Parameters
prefixDesired metrix prefix for the output unit (default = "" is barns, "m" is millibarns, etc.)
printTotalIf true, prints the total cross section to the screen.
Note
This function calculates the run-by-run cross sections for each run that was part of the BatchCalculate() call to the BeamNorm pointer passed to the constructor. The total cross section is the weighted average of these run-by-run measurements.
