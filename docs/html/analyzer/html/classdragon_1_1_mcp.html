void dragon::Mcp::calculate ( )

Calibrate ADC/TDC signals, calculate x and y positions.

Calibrates anode, tcal, and tac values; calculates x- and y-positions.

Note
Position calculation algorithm taken from the MSc thesis of Michael Lamey, Simon Fraser University, 2001, p. 25; available online at dragon.triumf.ca/docs/Lamey_thesis.pdf
